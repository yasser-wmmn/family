# 🚀 تطبيق مستكشف الأبطال (عائلة مشعل) - Boy Explorer

[![Version](https://img.shields.io/badge/Version-2.0-blue.svg)](https://github.com/yasser-wmmn)
[![Category](https://img.shields.io/badge/Kids-Education-orange.svg)](https://github.com/yasser-wmmn)

تطبيق ويب تفاعلي تعليمي متطور مصمم خصيصاً للأبطال (مشعل، محمد، ونايف)، يجمع بين التعلم، اللعب، وإدارة المهام العائلية بأسلوب "مستكشف الفضاء" والمستقبل.

## ✨ الجديد في النسخة المطورة (V2.0)
تم تحويل المنصة بالكامل لتصبح أكثر حداثة وذكاءً:
- **نظام النجوم الذكي:** اجمع النجوم عند إتمام المهام أو النجاح في الألعاب! 🌟
- **تصميم Glassmorphism:** واجهة "آبل" العصرية مع خلفيات زرقاء متدرجة وتأثيرات بصرية مذهلة.
- **تحديات بدنية:** قسم رياضي متكامل بصور توضيحية لتشجيع الأطفال على الحركة.
- **البرمجة للصغار:** لعبة برمجة منطقية تتكون من 5 مستويات مع عوائق ذكية.

## 🕹️ الأقسام والوحدات التعليمية
### 1. مغامرة البرمجة 💻
- **التحدي:** ساعد الروبوت للوصول إلى النجمة.
- **المهارة:** تعلم تفكير الخوارزميات والمنطق البرمجي.
- **التدرج:** تبدأ بسيطة ثم تظهر جدران وعوائق (🧱) في المستويات المتقدمة.

### 2. تعلم اللغة الإنجليزية (English Learner) 🇺🇸
- التعرف على الحروف الكبيرة (Capital Letters) وكتابتها بأسلوب تفاعلي.

### 3. التربية الإسلامية 🕌
- عداد الأذكار التفاعلي لغرس القيم الروحية بضغطة زر.

### 4. صيد النجوم الفضائية (Space Star Catcher) 🛰️
- لعبة سرعة بديهة مسلية لصيد الأجرام السماوية والألماس قبل انتهاء الوقت.

### 5. الرياضيات واللغات 🔢📖
- **الحساب:** تحديات جمع وطرح بأسلوب مبسط.
- **الحروف العربية:** التعرف على الحروف وأسمائها.
- **عالم الحيوان:** استكشاف الحيوانات وأسمائها.

## 👨‍👩‍👦‍👦 أفراد العائلة وطموحاتهم
- **أبو مشعل:** مهندس برمجيات محترف 💻.
- **أم مشعل:** خبيرة موارد بشرية وإدارة (Expert) 👑.
- **مشعل (الطيار):** يحلم بالتحليق في السماء ✈️.
- **محمد (رائد الفضاء):** يستعد لاستكشاف المجرات 🚀.
- **نايف (الدكتور):** يطمح لعلاج المرضى وإنقاذ الأرواح 🩺.

## 🛠️ التقنيات المستخدمة
- **Frontend:** Vanilla JavaScript, CSS3 (Modern Glassmorphism), HTML5.
- **Animation:** CSS Keyframes, Canvas-Confetti API.
- **Storage:** LocalStorage لحفظ النجوم والمهام.
- **Design:** Apple-style aesthetics, Cairo & Outfit Fonts.

## 🚀 كيفية البدء
1. قم بتحميل الملفات.
2. افتح `index.html` في أي متصفح حديث.
3. ابدأ رحلة الاستكشاف!
